# Green-House-Monitoring

This project helps you monitor the temperature,soil moisture,humidity and light intensity of a particular area where the module is placed and thus take actions accordingly.

1.You will need :
-Arduino Uno
-DHT 11 Temperature sensor
-Soil Mositure sensor
-LDR
-LCD 

2.Compile and upload the code in the microcontroller.
3.Connect the sensors at respective pins of arduino.
4.Run the code and monitor readings on LCD.
